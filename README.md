# Semantica
Words Meaning Calculator: perform vector math operation on words.

![](images/semantica.jpeg)

The logic of Semantica is built on the idea that if you can turn a word into a coordinate, you can treat human language like a map. Semantica is a "navigational" tool to explore this map.

# Description
The core logic uses [GloVe embeddings](https://nlp.stanford.edu/projects/glove/). Download `glove.6B.zip`, extract, put `glove.6B.300.txt` to the database folder.
